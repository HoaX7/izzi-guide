---
title: Filters
description: Filters help narrow down information from commands.
---
<Note>Almost all commands use similar filters.</Note>
## Inventory
- name `-n <card name>`
- series `-s <series>`
- rarity `-r <rarity>`
- element type `-t <type>`
- ability `-a <abilityname>`
- favorite `-fav`
- cards on market `-mk`
- cards on cooldown `-cd`

## Dex
Lists all characters on izzi.
- name `-n <card name>`
- series `-s <series>`
- element type `-t <type>`
- ability `-a <abilityname>`
- year `-year <year>`
- dark zone `-dz`
- monthly `-mn`
- custom card `-cc`
- world boss `-wb`

## Raid Lobby
- name `-n <card name>`
- series `-s <series>`
- rarity `-r <rarity>`
- element type `-t <type>`
- ability `-a <abilityname>`
- difficulty `-d <difficulty>`

## Global Market
- name `-n <card name>`
- series `-s <series>`
- rarity `-r <rarity>`
- element type `-t <type>`
- ability `-a <abilityname>`

<Note>Comma seperated values can be used. Ex: `iz market -n akane,rem,beerus -r immortal`.</Note>
